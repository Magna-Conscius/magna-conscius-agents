# Magna Conscius Agents

Public agent systems, orchestration patterns, and intelligence workflows from Magna Conscius.

## What This Repo Is
This repository is the public agent and intelligence-workflow layer of Magna Conscius.

It holds:
- agent registries
- workflow structures
- training and experiment-tracking notes
- public intelligence-system mapping

## Structure
- `registry/` - agent definitions and responsibilities
- `training/` - Colab workflow and experiment-tracking notes
- `workflows/` - public workflow and system-mapping material

## Purpose
The goal is to show how research, models, and systems can be turned into usable intelligence workflows rather than staying trapped as static documents.

## Related Repositories
- `magna-conscius` - private foundation repo
- `magna-conscius-web` - public implementation layer
- `magna-conscius-models` - public modeling layer
- `magna-conscius-research` - public research layer
